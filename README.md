# xylofyai-housing-project

Housing price analysis and prediction project developed during my internship at Xylofy AI.

## Description
An exploratory data analysis and machine learning project analyzing the impact of various real estate features (such as location, size, and amenities) on housing market prices. The dataset includes housing prices, area, bedrooms, bathrooms, and other features.

## Files
- `Housing.csv`: The housing dataset.
- `analysis.ipynb`: Jupyter notebook containing the analysis.
- `summary.docx`: Summary report of the project.
- `charts/`: Visualizations generated during analysis.
